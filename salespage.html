Features

Everything You Need for Hands-Free Outreach

Built for marketers who want results without the repetitive manual work.

📅

Daily Message Scheduling

Assign unique messages to specific dates. Plan your entire content calendar in advance and the extension sends the right message on the right day — automatically.

⚡

Smart Batch Randomization

Set min/max batch sizes and the extension randomizes within your range each cycle. No two batches look the same — keeping your activity pattern natural.

🕐

Human-Like Timing

Randomized delays between each message and between cycles simulate real human behavior. Configurable sleep intervals from 30 to 90+ minutes.

🌎

Location Filtering

Define restricted regions and the extension automatically scans profiles and skips viewers from locations you don't want to target. Precision outreach, zero waste.

🔒

Duplicate Prevention

Every contacted user is stored in memory — across sessions and cycles. The extension will never message someone twice, even if they view multiple stories.

📊

Live Stats Dashboard

Track messages sent, viewers skipped, and next batch timing in real time. Daily counters reset automatically so you always know today's performance.
