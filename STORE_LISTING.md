# Chrome Web Store Listing Details

## Name
Insta Outreach Automator

## Short Description (132 chars max)
Schedule and automate personalized Instagram story viewer outreach with smart batching and daily message scheduling.

## Detailed Description
Insta Outreach Automator helps you connect with your Instagram story viewers through personalized, scheduled direct messages.

KEY FEATURES:

- Scheduled Messages: Plan your outreach calendar by assigning unique messages to specific dates. When a scheduled message exists for today, it's used automatically. Otherwise, your default template kicks in.

- Smart Batching: Set minimum and maximum batch sizes. The extension randomizes within your range each cycle to keep outreach natural.

- Human-Like Timing: Built-in randomized delays between messages and between outreach cycles simulate natural behavior patterns.

- Location Filtering: Specify restricted locations to automatically skip viewers from regions you don't want to target.

- Duplicate Prevention: The extension remembers every user you've already messaged and never contacts them twice — even across sessions.

- Daily Stats Dashboard: Track how many messages were sent and skipped today, plus see when the next batch is scheduled.

- Import Memory Database: Migrating from another tool? Paste your existing contacted-users list to avoid duplicates.

HOW IT WORKS:

1. Enter your Instagram username and set your default message template.
2. Optionally schedule date-specific messages for your content calendar.
3. Configure batch sizes, sleep intervals, and restricted locations.
4. Hit "Start Campaign" — the extension handles the rest.

The extension opens your Instagram profile, scrapes your story viewers, filters out previously contacted users and restricted locations, then sends personalized DMs in randomized batches with natural delays.
